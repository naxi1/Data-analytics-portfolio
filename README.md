This repository contains a series of end-to-end data analytics projects focused on extracting business insights from retail transaction data. The projects demonstrate practical skills in data cleaning, exploratory data analysis, customer segmentation, SQL-based data pipelines, and pricing optimization.
⸻
Projects Overview

Project 1 — Exploratory Data Analysis (EDA)
	•	Cleaned and explored raw retail dataset
	•	Identified sales patterns and key business metrics
	•	Built foundational understanding of data structure
⸻

Project 2 — Customer Segmentation (K-Means Clustering)
	•	Grouped customers based on behavior
	•	Identified high-value, regular, and loss-making customers
	•	Used clustering to support targeted business decisions
⸻

Project 3 — Inventory & Product Analysis
	•	Analyzed product performance
	•	Identified best and worst-selling products
	•	Highlighted profit and inventory inefficiencies
⸻

Project 4 — SQL Business Analysis
	•	Stored dataset in SQL database
	•	Performed category and trend analysis using SQL queries
	•	Extracted business insights using structured queries
⸻

Project 5 — Dashboard (Power BI)
	•	Will include interactive business dashboard using Excel or Power BI
⸻

Project 6 — End-to-End Data Pipeline
	•	Built full data workflow from raw data to insights
	•	Created customer, product, and category views using SQL
	•	Simulated real-world analytics pipeline
⸻

Project 7 — Pricing & Profit Optimization
	•	Analyzed impact of discounting on profitability
	•	Identified optimal discount range
	•	Provided recommendations for pricing strategy
⸻
